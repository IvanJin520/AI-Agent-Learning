🎯 课后巩固题
第 1 题：Token 在 AI 中的作用是什么？
  • A. 代表模型的权重
  • B. 用来计费的单位，也是模型处理的基本单位
  • C. 代表 AI 的智力水平
  • D. 用来存储用户信息

✅ 正确答案：B

Token 是 AI 模型处理信息的最小单位。当你和 AI 交互时，你的输入和 AI 的输出都会被转换成 Token，然后被模型处理。API 服务商根据消耗的 Token 数量来计费。

第 2 题：你说的 800 万 Token 大约相当于多少汉字？
  • A. 400 万
  • B. 1600 万
  • C. 800 万-1200 万
  • D. 200 万

✅ 正确答案：C（800 万-1200 万）

中文是个比较复杂的语言。一个汉字通常需要 1-2 个 Token 来表示。这就是为什么你用 AI 问了几个问题就花了 800 万 Token。

第 3 题：什么是大语言模型（LLM）？用你自己的话解释（开放题）

📝 参考答案：

大语言模型（LLM）是一种 AI 模型，经过大量文本数据的训练，能够理解和生成人类语言。它的"大"指的是模型的参数数量巨大（通常在十亿到数千亿）。LLM 的核心能力是：根据你给它的输入（提示词），预测和生成最可能的下一个 Token，从而形成有逻辑的输出。

第 4 题：Agent 和 ChatGPT 最根本的区别是什么？
  • A. Agent 更聪明
  • B. ChatGPT 是免费的
  • C. Agent 可以使用工具，并根据结果反复思考和调整
  • D. Agent 只能用中文

✅ 正确答案：C

Agent 不仅能理解你的目标，还能决定要做什么、使用工具来执行、根据结果反馈继续调整，直到完成目标为止。每一步都在消耗 Token，这就是为什�� Agent 更贵。

第 5 题：如果一个 Agent 需要思考 5 次才能完成一个任务，而每次思考消耗 5000 Token，那么一个任务的成本是多少 Token？
  • A. 5000
  • B. 25000
  • C. 50000
  • D. 无法确定

✅ 正确答案：B（25000）

计算：5 次思考 × 每次 5000 Token = 25000 Token。这就是为什么 Agent 这么贵。如果你能把"5 次思考"优化成"2 次思考"，成本就能降低 60%！

第 6 题：如果你用 OpenAI GPT-4 的 Agent 做报价，一个月处理 1000 次，每次 25000 Token。费用是多少？

✅ 答案计算：

1000 次 × 25000 Token ÷ 1000 × ¥0.03 = ¥750

一个月 ¥750，一年就是 ¥9000！

第 7 题（关键）：如果改用国内模型（火山引擎，¥0.001/千 Token），同样的任务费用是多少？便宜多少倍？

✅ 答案计算：

1000 次 × 25000 Token ÷ 1000 × ¥0.001 = ¥25

相比 OpenAI 便宜：¥750 ÷ ¥25 = 30 倍

同样的功能，用国内模型只需要 ¥300 一年，用 OpenAI 需要 ¥9000 一年！
